# 畫地色線
## 簡介

不同於傳統桌遊，「畫地色限」是一款融合虛擬與現實的戰略型爭地桌遊。使用實體卡牌並在數位化的畫板上進行遊戲，有著9種角色可供選擇，每個角色各有不同的技能，還有千變萬化的道具卡，擴展自己或是讓別人失去領土。遊玩人數可以3到6人。
